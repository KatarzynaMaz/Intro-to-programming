Inline vs block elements

Inline elements

Inline element does not start on a new line and only takes as much width as necessary. Examples include:

  • br
  • img
  • span
  • mark
  • em
  • a
  • em
  • strong
  • b

Block elements

A block alement starts on a new line. It occupies the whole horizontal space/width of its parent element and vertical space equal to the height of its contents, so it creates a block around itself. Examples include:

  • div
  • footer
  • main
  • section
  • article
  • aside
  • table
  • p

A p element tells the browser to create a little block around the element.

There are a few HTML elements that always apprear inside other elements, otherwise they don't make sense. A list element is a good example. A list element li always aprears inside the list (ordered or unordered). Lists, however, can be nested inside each other.
